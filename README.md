# Manaita Successor Mod

This is a complete remake of the popular Minecraft 1.7.10-era mod "Manaita".

This mod adds Manaita-themed blocks, progression crafting, and powerful crafting multipliers that scale with each Manaita tier.

## Features

- 5 Manaita tiers: `wooden`, `stone`, `iron`, `gold`, `diamond`
- Crafting multipliers by tier:
  - `wooden`: `2x`
  - `stone`: `4x`
  - `iron`: `8x`
  - `gold`: `16x`
  - `diamond`: `32x`
- Extra item: `plate`
- Legacy-style progression recipes from lower to higher board tiers
- Configurable drop replacement to `plate` (`plateDropChance`, default `0.125`)

## How to Obtain `plate` 

- Break `Oak Planks` blocks in the world.
- When `plateDropChance` succeeds (default `0.125`), the normal Oak Planks drop is replaced with `plate`.

## Current Target

- Minecraft `1.21.11`
- Fabric Loader

## Notes

The project is designed as a maintainable successor implementation inspired by the original gameplay feel, not a direct copy of decompiled legacy code.
