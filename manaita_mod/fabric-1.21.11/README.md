# Manaita Fabric 1.21.11

## What This Module Contains

- Wooden/Stone/Iron/Gold/Diamond cutting boards (`2x/4x/8x/16x/32x`)
- `plate` item and legacy-like board progression recipes
- Oak planks break chance replacement to `plate` via config

## Commands

- Build: `./gradlew.bat build`
- Run client: `./gradlew.bat runClient`

## Config

Generated at first launch:

- `config/manaita.json`
  - `plateDropChance` (default: `0.125`)
  - `enableCheat` (reserved, currently unused)
